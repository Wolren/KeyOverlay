# KeyOverlay

A WPF overlay application for tracking and visualizing keyboard and mouse input in real time — useful for measuring click speed and key activity on top of other apps.

## Features

- **Keyboard Overlay** — displays live keyboard input
- **Mouse Overlay** — tracks and displays mouse button presses
- **Key Overlay** — fully customizable key tracking overlay
